    Introduction

    Welcome to the Jobhuk API! You can use our API to access Jobhuk API endpoints, you publish your jobs in our website and get candidates.

    We have language bindings in Shell and Ruby! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Authentication

    To authorize endpoint urls, example:

    require 'httparty'

response = HTTParty.get('http://api.jobhuk.com/companies.json', headers: {"Token" => "API_TOKEN"})
companies = response.body

    curl "http://api.jobhuk.com/companies.json"
  -H "Token: API_TOKEN"

    Make sure to replace API_TOKEN with your API key.

    Jobhuk uses API keys to allow access to the API. Get API TOKEN.

    Jobhuk expects for the API_TOKEN to be included in all API requests to the server in a header that looks like the following:

    Token: API_TOKEN

    ACCEPT: application/vnd.jobhuk.v1

    You must replace `API_TOKEN` with your personal API key.

    This endpoint post a job in jobhuk.

    HTTP Request

    POST http://api.jobhuk.com/jobs.json

    POST Parameters

    Parameter Value Description
    job
    required    		 hash job fields hash
    job[company_id]
    required    		 integer company id for job
    job[title]
    required    		 string title of the job
    job[description]
    required    		 string description of the job
    job[location]
    required    		 string location of the job
    job[job_type]
    required    		 string type of the job
    Must be one of FullTime, Contract.
    job[comp_range]
    optional    		 integer compensation range in percentage
    job[compensation]
    required for fulltime jobs    		 integer candidate compensation for job
    job[finders_fee_type]
    required for fulltime jobs    		 string finders fee type Must be one of percentage, fixed.
    job[finders_fee]
    required for fulltime jobs    		 integer finders fee percentage if finders fee type is percentage
    job[finders_fee_amount]
    required for fulltime jobs    		 integer finders fee amount if finders fee type is fixed
    job[duration_type]
    required for contract jobs    		 string Must be one of Hours, Weeks, Months.
    job[duration]
    required for contract jobs    		 integer job duration like 320 hours or 4 weeks or 2 months
    job[hourly_rate]
    required for contract jobs    		 integer hourly rate for contract jobs.
    job[active]
    optional    		 boolean job active for candidates to apply.
    job[show_market_place]
    optional    		 boolean true showing jobhuk market place.
    job[joining_date]
    optional    		 string candidate joining date Must be yyyy-mm-dd format.
    job[travel_percentage]
    optional    		 integer candidate relocation charges percentage
    job[relocation]
    optional    		 boolean is candidate ready to relocation for job
    job[payment_duration]
    optional    		 integer finders fee payment duration in days
    job[client_name]
    optional    		 string client company name for this job if required
    job[skills]
    optional    		 string list skills required for this job comma separated.
    job[industry]
    optional    		 string job industry type

    Errors

    The Jobhuk API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slown down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
